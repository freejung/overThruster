--------------------
overThruster
--------------------
Version: 0.1
Author: Eli Snyder <freejung@gmail.com>
--------------------

A MODX Revolution extra that improves uncached performance by storing the output of listing snippets (wayfinder and getResources) in custom db tables for quick retrieval.

Feel free to suggest ideas/improvements/bugs on GitHub:
http://github.com/freejung/overThruster/issues
